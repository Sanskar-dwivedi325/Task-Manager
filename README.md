# Project Workspace

A full-stack project and task management platform with role-based access, built using Spring Boot (backend) and React (frontend).

---

## 🚀 Deployment

- **Frontend:** Vercel  
- **Backend:** Render  
- **Database:** Aiven (MySQL-compatible cloud database)

---

## 🧑‍💻 How to Use

### 1. Login / Signup
- Open the frontend link.
- Login using demo credentials or create a new account.

---

### 2. Admin Workflow
- Create new projects  
- Add or remove members  
- Assign tasks to members  
- View all project tasks  
- Delete projects or tasks  
- Access dashboard metrics (total, completed, pending, overdue)

---

### 3. Member Workflow
- View assigned projects  
- View assigned tasks  
- Update task status:
  - TODO  
  - IN_PROGRESS  
  - DONE  
- Track personal dashboard metrics  

---

### 4. Task Flow
1. Admin creates a project  
2. Admin adds members  
3. Admin assigns tasks  
4. Members update task status  
5. Dashboard updates automatically  

---

## ✨ Features

- JWT-based authentication  
- Role-based access control (Admin / Member)  
- Project management  
- Task assignment and tracking  
- Dashboard analytics  
- Secure REST APIs  
- Responsive UI
